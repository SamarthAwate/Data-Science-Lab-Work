Description

The objective of this project is to create an interactive text-based adventure game using Python, where the player navigates through a city at night and makes decisions that affect the outcome of the story. The project demonstrates basic programming concepts such as conditional statements, loops, and user interaction.

Dataset
Source: Self-created story
Description of data:
The “data” in this project consists of a custom-designed storyline with multiple branches, choices, and possible endings. Random events are generated using Python’s built-in random module.

Steps Performed
Data Creation
Designed a multi-branch storyline with different paths (cafe, alley, subway).
Logic Implementation
Used if-else statements to control story flow based on user choices.
Interactivity
Implemented input() to allow user decisions during gameplay.
Randomization
Used the random module to introduce unpredictable events.

Results
The game successfully allows users to:
Make decisions that impact the story
Experience different outcomes (win/lose)
Each playthrough can be different due to random events

Tools Used
Python
Built-in Libraries:
time
random

Conclusion

This project demonstrates how basic Python programming concepts can be used to build an interactive application. It highlights the use of logic, functions, and randomness to create engaging user experiences. The project can be further expanded with more features such as inventory systems, health tracking, and graphical interfaces.
